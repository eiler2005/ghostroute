# Documentation Archive

This directory keeps dated research notes, handoffs, and roadmaps that are useful
as implementation history but are not current operator guidance.

Current production guidance lives in:

- `README.md` and `README-ru.md`
- `docs/architecture.md`
- `docs/operational-modules.md`
- `docs/troubleshooting.md`
- `docs/adr/`

Archive areas:

- `channel-c/` - historical Channel C client/debug research. Channel C remains a
  future experimental manual device-client lane.
- `roadmaps/` - dated improvement roadmaps kept for traceability after the main
  architecture docs were updated.
